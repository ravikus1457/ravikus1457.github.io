Privacy Policy — Symbio Radar

Effective: July 9, 2026

Symbio Radar is a personal, single-user tool operated by its developer to upload original videos to the developer's own YouTube channel and read analytics for that same channel via YouTube API Services.

Data we access

The tool accesses only the developer's own channel data: video upload capability and channel analytics (views, likes, watch time). It does not access, collect, or store any information about viewers, subscribers, or any other person.

Data we store

Video metadata and aggregate analytics for the developer's own videos, stored locally on the developer's private device. No personal information about any third party is collected, stored, shared, or sold.

Third parties

No data is shared with, sold to, or processed by any third party.

Google user data

Authentication uses Google OAuth. Access can be revoked at any time at myaccount.google.com/permissions. Use of YouTube API Services is subject to the YouTube Terms of Service and the Google Privacy Policy.

Data retention & deletion

The only stored authorized data is the operator's own OAuth token and metadata for the operator's own videos, kept locally on the operator's private device. Revoking access via the link above immediately invalidates the stored token; locally stored tokens and metadata are deleted on request or when the tool is retired, and can be removed at any time by contacting ravikus1457@gmail.com. No other user's data is ever stored, so there is no third-party data to retain or delete.
